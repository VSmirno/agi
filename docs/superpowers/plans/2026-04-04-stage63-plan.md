# Stage 63: Implementation Plan

**Spec:** `docs/superpowers/specs/2026-04-04-stage63-abstraction-crafter-design.md`

## Tasks

### Task 1: Abstraction Engine
**File:** `src/snks/agent/abstraction_engine.py`

1. `discover_categories(neocortex_rules)` — scan rules, group objects by (action, outcome)
2. Handle multi-category objects (key = carryable + solid + activator)
3. `encode_abstract_rule(codebook, category, action)` → VSA vector
4. `build_abstract_sdm(categories, codebook)` → SDM with ~20-50 rules
5. Verify: categories discovered correctly from Stage 62 neocortex

### Task 2: CLSWorldModel v2 — Pure SDM Generalization
**File:** `src/snks/agent/cls_world_model.py`

1. Remove neocortex color substitution from `query()` and `query_reward()`
2. Add `abstract_query(obj_type, action)` using abstraction engine
3. Query flow: neocortex → abstract SDM → hippocampus (fallback)
4. Same-color encoding: `bind(color_X, color_X) = zero` in abstract rules
5. Verify on minipc GPU: held-out colors ≥85% without substitution

### Task 3: Object-Type Hold-Out Tests
**File:** `tests/test_stage63_abstraction.py`

1. Train on key+door only (remove ball+box from synthetic)
2. Test: "can you pickup ball?" → should generalize via "carryable" category
3. Test: "forward into box?" → should generalize via "solid" category
4. Gate: ≥80% on held-out object types
