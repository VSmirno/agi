Как это работает

Идея: модель учится из 5 коротких демонстраций (3-5 шагов каждая), как младенец учится от родителя. Каждое правило хранится в отдельной SDM (Sparse Distributed Memory) через VSA-кодирование.

Ключевое свойство: bind(X, X) = zero_vector для любого цвета X. Это значит, что пара "красный ключ + красная дверь" и "зелёный ключ + зелёная дверь" дают одинаковый адрес в SDM (нулевой вектор). Поэтому правило, выученное на красном цвете, автоматически работает для зелёного, фиолетового или любого другого — без единого примера этих цветов.

Почему это важно для AGI: каузальное обобщение из минимальных данных — фундамент для world model, способной рассуждать о новых ситуациях. Не нужен миллион примеров, достаточно 3 демонстраций.

Формула identity

bind(red, red) = zero_vector
bind(blue, blue) = zero_vector
bind(green, green) = zero_vector ← unseen!

bind(red, blue) = random_vector_1 ≠ zero
bind(red, green) = random_vector_2 ≠ zero

SDM[zero_vector] = reward +1 → "одинаковый цвет = открывает"
