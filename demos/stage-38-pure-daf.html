Как это работает

Полный цикл PureDafAgent (без scaffolding):

1. Наблюдение → изображение из камеры агента (7×7×3 пикселей)
2. Кодирование → Gabor-фильтры → SDR (разреженное распределённое представление)
3. Осцилляторы → SDR инжектируется как ток в 50,000 FHN-нейронов → 100 шагов интеграции
4. Когерентность → синхронные группы нейронов = SKS кластеры (концепты)
5. Встраивание → HAC-вектор (2048-мерный) для каждого SKS
6. Выбор действия → ментальная симуляция: для каждого действия предсказывается следующее состояние
7. STDP обучение → spike-timing-dependent plasticity модифицирует связи
8. Reward-модуляция → положительная награда усиливает недавние STDP изменения

Каузальное обучение: вместо словаря (context, action) → effect, каузальные связи закодированы в весах 2.5M синаптических связей. Каждое действие создаёт моторный ток → изменяет динамику → STDP усиливает связи между ко-активными нейронами → формирует action-conditioned аттракторы.
Почему это важно для AGI:

Stages 19-37 доказали, что СНКС может решать задачи (goal decomposition, transfer, analogy, planning). Но решал их символический planner (BFS + dict), обёрнутый в осцилляторную обёртку. Stage 38 замыкает петлю: теперь вычисление = эволюция динамической системы. Осцилляторы формируют концепты, STDP модифицирует связи, reward модулирует обучение. Результаты хуже (12% vs 100%), но это настоящие результаты нейроморфной системы без backpropagation.

Архитектура: Environment-Agnostic

PureDafAgent (не зависит от среды)
├── CausalAgent
│   └── Pipeline → DafEngine (50K FHN, STDP, гомеостаз)
│       ├── VisualEncoder: image → Gabor → SDR → токи
│       ├── SKS Detection: когерентность → DBSCAN кластеры
│       └── HAC Prediction: встраивание → предсказание
├── DafCausalModel (reward-modulated STDP)
│   ├── Трасса: snapshot весов → действие → награда → модуляция
│   └── Ментальная симуляция: inject state + action → predict
├── AttractorNavigator (goal-directed)
│   ├── Для каждого действия: predict → cosine similarity к цели
│   └── Выбор: argmax similarity (или exploration)
└── EnvAdapter (протокол)
    ├── MiniGridAdapter: gymnasium MiniGrid → obs/action
    ├── ArrayEnvAdapter: любой array-env → pseudo-image
    └── Нет grid.get(), нет unwrapped, нет hardcoded SKS
