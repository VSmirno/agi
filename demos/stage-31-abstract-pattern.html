How It Works

Raven's Progressive Matrices are a standard IQ test: find the missing piece in a 3x3 grid. Human solvers discover transformation rules (each row applies the same change).

СНКС solves this using HAC algebra — the same holographic codes that store concepts (SKS). Each cell is an SKS embedding (2048D vector). The reasoner:

  1. Extracts transforms: T = unbind(cell_i, cell_{i+1}) — what changed?
  2. Checks consistency: Is T the same across all rows? (cosine similarity)
  3. Predicts missing: answer = bind(last_known, T) — apply the rule
  4. Selects answer: pick the option closest to prediction (cosine similarity)

No neural network, no backpropagation, no training. Pure algebraic reasoning on concept vectors. This is the same bind/unbind used for memory (Stage 9), prediction (Stage 10), and skills (Stage 27).
