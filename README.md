# СНКС — Система Непрерывного Когнитивного Синтеза

> Экспериментальная архитектура AGI, принципиально отличающаяся от LLM и классических нейросетей.

**Статус:** Stages 0–30 COMPLETE · 75 экспериментов PASS · MVP доказан

---

## Идея

Современные нейросети (трансформеры, CNN, RNN) и LLM — это функции, преобразующие вектора: вход проходит через слои, умножаются матрицы, применяется нелинейность, backpropagation корректирует веса. Этот подход принципиально ограничен: он требует огромных датасетов, страдает от catastrophic forgetting, а «мышление» сводится к генерации токенов — не к оперированию смыслами.

**СНКС строится иначе.** Система моделирует мозг как **динамическую систему связанных осцилляторов**, которая самоорганизуется и формирует устойчивые паттерны — аттракторы. Знание — это не вес в матрице, а стабильная синхронизация группы осцилляторов. Обучение — не глобальный градиентный спуск, а локальные биологически достоверные правила (STDP).

Главный принцип: **концепты первичны, язык вторичен**. Слово — это якорь, привязанный к уже существующей концептной структуре. Мышление происходит в пространстве концептов, без перевода на человеческий язык.

---

## Ключевые отличия от классического ML

| | Классические нейросети | СНКС |
|---|---|---|
| **Обучение** | Backpropagation, глобальная оптимизация | Локальные правила STDP, нет глобального градиента |
| **Представления** | Статические вектора (эмбеддинги) | Устойчивые колебательные паттерны (аттракторы) |
| **Память** | Веса сети = вся память | HAC + SSG: содержание отдельно от структуры связей |
| **Непрерывное обучение** | Catastrophic forgetting | Новые знания не стирают старые (retention > 100%) |
| **Вычисления** | Прямой проход: вход → выход | Эволюция динамической системы к аттрактору |
| **Язык** | Основа мышления (токены) | Интерфейс ввода-вывода к концептному пространству |

## Архитектура

```
Сенсорный вход (изображение / текст / действие)
       │
       ▼
┌─────────────────────────┐
│  Visual/Text Encoder     │  Биологическая модель восприятия
│                          │  Результат: SDR (разреженный двоичный вектор)
└────────────┬────────────┘
             │
             ▼
┌─────────────────────────┐
│  ДАП-Движок (DAF)        │  10 000+ осцилляторов FitzHugh-Nagumo
│  Dynamic Attractor Field │  Эволюция к аттрактору: стимул → самоорганизация
│                          │  STDP обучение + гомеостатическая регуляция
└────────────┬────────────┘
             │
             ▼
┌─────────────────────────┐
│  СКС-детектор            │  Стабильные Когнитивные Структуры
│  Stable Cognitive Struct │  Кластеры осцилляторов = «концепты»
└────────────┬────────────┘
             │
             ▼
┌─────────────────────────┐
│  World Model (DCAM)      │  HAC: голографические векторы содержания
│                          │  SSG: разреженный граф структуры связей
│                          │  Consolidation (аналог сна)
└─────────────────────────┘
             │
    ┌────────┴────────┐
    ▼                 ▼
┌────────┐      ┌──────────────┐
│ GoalAgent│    │  Verbalizer  │
│Planning  │    │ Концепт→текст│
│Curiosity │    │ Grounded QA  │
└────────┘      └──────────────┘
```

### Ключевые компоненты

- **ДАП (Dynamic Attractor Field)** — поле динамических аттракторов. Тысячи связанных нелинейных осцилляторов, эволюционирующих к устойчивым состояниям под воздействием входного сигнала.
- **СКС (Стабильные Когнитивные Структуры)** — концепты. Группы осцилляторов, стабильно синхронизирующихся при определённом стимуле. Аналог «понятия» в мозге.
- **DCAM (Dual-Code Associative Memory)** — двухкодовое ассоциативное хранилище мира: HAC для содержания + SSG для структуры связей.
- **GoalAgent** — агент с декомпозицией целей, backward chaining, каузальным обучением и любопытством.
- **GroundedTokenizer / Verbalizer** — языковой интерфейс: текст ↔ концептное пространство без внешних LLM.

## Философия проекта

Андрей Карпатый точно сформулировал проблему: мы переводим суть вещей в человеческий язык, чтобы оперировать им, а потом переводим обратно. Это подход с огромным оверхедом. Познание должно происходить без перевода на язык.

СНКС — попытка построить систему, которая:
- Думает концептами, а не токенами
- Учится локально, без глобального учителя
- Не забывает старое при изучении нового
- Исследует мир из любопытства, а не ради reward
- Использует язык как инструмент, а не как субстрат мышления

Это не production-система. Это исследовательский проект, проверяющий гипотезу: **можно ли построить AGI без backpropagation**.
