# Этап 5: Визуализация + Интеграция

**Статус:** ✅ Завершён
**Срок:** Неделя 8–10
**Зависимости:** Этапы 3 ✅, 4 ✅

## Цель

End-to-end pipeline с real-time визуализацией и Экспериментом 5 (персистентность).

## Модули

| Модуль | Назначение |
|--------|-----------|
| `viz/server.py` | FastAPI + WebSocket сервер |
| `viz/static/` | D3.js frontend для визуализации ДАП, СКС, DCAM |
| `experiments/exp5_persistence.py` | Exp 5: save → reload → retest |

## Функциональность дашборда

- Граф осцилляторов (top-K активных) — цвет = фаза/амплитуда
- СКС кластеры — визуализация в реальном времени
- DCAM граф — SSG слои, эпизоды
- Метрики: NMI, PE, firing rate, cycle time
- Управление: запуск/пауза, подача стимулов

## Gate

- Полный pipeline end-to-end
- Dashboard FPS ≥ 5
- Эксперимент 5: Δ accuracy ≤ 1% после save/load
